# Biya Kormu

**Biya Kormu** is a modern matrimonial platform built with **Django REST Framework** and **React**. It helps users find compatible life partners through secure authentication, detailed profiles, advanced search, premium memberships, and a seamless user experience.

## ✨ Features

- 💍 **Matrimonial Profiles** – Create and manage detailed personal profiles.
- ❤️ **Partner Search** – Find suitable matches using advanced search and filters.
- 🔐 **Authentication & Security** – JWT authentication with OTP verification.
- 👤 **Profile Management** – Update personal information, preferences, and profile photos.
- 💎 **Premium Membership** – Unlock premium features with membership plans.
- 💳 **SSLCommerz Payments** – Secure online payment integration.
- 📸 **Cloudinary Uploads** – Optimized cloud-based image storage.
- 📱 **Responsive Design** – Fully optimized for desktop, tablet, and mobile devices.

---

## 🛠 Tech Stack

- **Frontend:** React.js, Vite, Redux Toolkit, Tailwind CSS, React Router, Radix UI
- **Backend:** Django, Django REST Framework
- **Database:** SQLite (Development)
- **Authentication:** JWT, OTP Authentication
- **Caching:** Redis
- **File Storage:** Cloudinary
- **Payments:** SSLCommerz
- **API Documentation:** Swagger (DRF-YASG)

---

## 🔧 Admin Features

- 👥 **User Management** – View, manage, verify, and remove user accounts.
- 📋 **Profile Moderation** – Review and manage matrimonial profiles.
- 💎 **Membership Management** – Manage premium subscriptions.
- 💳 **Payment Management** – Monitor SSLCommerz payment transactions.
- 📊 **Dashboard Analytics** – Track users, memberships, and platform activity.
- 🔐 **Role-Based Access Control** – Secure admin dashboard with protected routes.
